Mini App «Зв з гуртожитку» на InfinityFree (dash.infinityfree.com)
================================================================

Бот залишається на вашому VPS; на InfinityFree потрібна лише одна статична
сторінка — файл webapp/zv_dorm_form.html з цього репозиторію.

1) Акаунт і сайт
   - Створіть сайт у панелі InfinityFree (або використайте вже створений).

2) Підключення домену
   - Якщо використовуєте свій піддомен (наприклад c55scheluderzv.gamer.gd),
     додайте його в InfinityFree згідно їхньої інструкції (DNS: зазвичай
     CNAME на їхній хост або зміна NS у реєстратора — як вимагає панель).
   - Дочекайтесь, поки SSL стане активним (у панелі часто є статус HTTPS).

3) Завантаження файлу
   - File Manager або FTP → каталог public сайту (часто htdocs).
   - Завантажте zv_dorm_form.html у корінь сайту (або в підкаталог — тоді
     URL буде з шляхом, наприклад /zv/zv_dorm_form.html).

4) Перевірка в браузері
   - Відкрийте повний HTTPS-URL файлу; має відкритися форма без помилки SSL.

5) Бот (.env на VPS)
   - ZV_DORM_WEBAPP_URL=<той самий повний URL, що в браузері>
     Приклад: https://c55scheluderzv.gamer.gd/zv_dorm_form.html
     або:    https://ваш-сайт.infinityfreeapp.com/zv_dorm_form.html
   - Перезапустіть бота після зміни .env.

6) Telegram / BotFather
   - Якщо клієнт просить «домен» для Web App — додайте домен згідно підказок
     для вашого хоста (без https://, зазвичай лише hostname).

Примітка: nginx на VPS для цього домену не потрібен, якщо DNS веде на
InfinityFree — тоді приклад scripts/nginx-zv-dorm-miniapp.example.conf
стосується лише варіанту «домен вказує на ваш VPS».
